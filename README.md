🍔 Projeto Lanchonete

Projeto experimental desenvolvido utilizando HTML, CSS e JavaScript, com foco em aprendizado prático de desenvolvimento web, estruturação de páginas e interatividade no navegador.

O sistema simula uma interface simples de lanchonete, permitindo explorar conceitos básicos de front-end, organização visual e manipulação de elementos utilizando JavaScript.

🚀 Tecnologias Utilizadas
HTML5
CSS3
JavaScript
🎯 Objetivo do Projeto

Este projeto foi criado apenas para fins de estudo e experimentação, servindo como prática de:

Estruturação de páginas web
Estilização com CSS
Manipulação do DOM com JavaScript
Criação de interfaces simples e responsivas
Organização de componentes visuais
📋 Funcionalidades
Interface visual de uma lanchonete
Exibição de produtos
Layout estilizado
Elementos interativos com JavaScript
Navegação simples e intuitiva

▶️ Como Executar

Clone o repositório:

git clone https://github.com/JoalissonAraujo/ProjetoLanchonete.git

Entre na pasta do projeto:

cd ProjetoLanchonete

Abra o arquivo index.html no navegador.

💡 Observações

Este projeto não possui integração com banco de dados ou back-end, sendo apenas uma aplicação front-end experimental.
